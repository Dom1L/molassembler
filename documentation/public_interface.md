# Public interface

## v1.0.0
All headers in the base lib directory of molassembler and files included from
there form the public interface. The auxiliary library headers that are
also part of the public interface are currently:

- chemical symmetries/
  - Names

- temple/
  - Preprocessor
  - Random
  - TinySet
  - Traits
  - constexpr/
    - Bitmask
    - JSF
    - Math
